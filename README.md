# LagMod

Eine Performance-Optimierungsmod für Minecraft 1.21.1 basierend auf Fabric.

## Installation

1. Stelle sicher, dass du Fabric Loader 0.16.7 oder höher installiert hast
2. Laden Sie die neueste Version von LagMod herunter
3. Platziere die JAR-Datei im `mods`-Ordner deines Minecraft-Verzeichnisses
4. Starten Sie Minecraft neu

## Kompilierung

```bash
./gradlew build
```

Die compilierte Mod befindet sich in `build/libs/lagmod-1.0.0.jar`

## Features

- Performance-Optimierungen
- Konfigurierbare Einstellungen
- Kompatibel mit anderen Mods
